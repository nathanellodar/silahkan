BACA DAN LAKUKAN SELURUH INSTRUKSI DI BAWAH INI DENGAN BAIK!

1. Rename file md dengan format NIM-No.md.  Misalnya 71230123-01.md jika NIM anda adalah 71230123
dan ini adalah laporan praktikum 01.  Gunakan penomoran 01, 02, 03, dst.... (dua digit)

2. Masukkan seluruh file yang dibutuhkan (image, source code, md, dsb...) dalam satu folder dengan 
format NIM-No.  Misalnya 71230123-01.  Kemudian zip folder tersebut sehingga menghasilkan file dengan 
nama 71230123.zip.  File inilah yang anda kumpulkan di LMS UKDW.

3. Penamaan file tersebut sangat penting sekali karena dibutuhkan pada saat pengecekan plagiasi, kelengkapan
dan kesesuaian format jawaban.  Jika tidak memenuhi syarat untuk tahap ini, maka pekerjaan yang
anda kumpulkan dianggap melanggar aturan dan tidak akan mendapatkan nilai.
